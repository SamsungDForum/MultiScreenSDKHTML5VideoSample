HTML5 Video with MultiScreen 2.0

MP4 files can be tested in Tizen Simulator. Other formats need TV.

Info:
PLAY - Start playing the video; STOP - Stop playing; 1 - change source;

The application uses websockets to communicate with the client application. If websockets are disabled in your local network the application will not work.

The application downloads video content from http://media.w3.org - it will not work if this domain is not accessible
You can change the source for video content by modifying contents of sources array in main.js file
